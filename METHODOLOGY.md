# Methodology & Principles

This project is a personal knowledge base documenting the development of the **Table Method**, a systematic and visual approach to English grammar.

## Core Philosophy
The Table Method shifts the focus from traditional dependency-based grammar analysis to a modular, table-oriented system. The goal is to create a more intuitive, scalable, and visual framework for understanding complex grammatical structures.

## AI Collaboration Disclosure
This project is developed through a human-led methodology. While Gemini (Google) is utilized for structural refinement, logical organization, and iterative drafting, the core pedagogical framework and final decision-making remain under the control of the human author. As the author’s own understanding of English grammar evolves, the AI serves as a collaborative partner in refining these concepts, rather than an absolute authority.

## Workflow & Principles of Use

### Development Workflow
This project is an iterative, "living" project. Development follows these principles:
1. **Human-Centric & Evolving:** All final logic is reviewed and validated by the author to the best of their current knowledge. As this project serves as a personal learning journey, the content reflects an evolving understanding of English grammar.
2. **Modular Structure:** Grammar rules are broken down into self-contained table components, allowing for flexible adoption and independent testing.
3. **Transparent Collaboration:** The use of AI as an architectural tool is fully disclosed to maintain project integrity.

### User Responsibility & Disclaimers
1. **Critical Engagement:** While the author strives for accuracy, this project is an experimental methodology. Users should exercise their own independent judgment and verify grammatical logic against authoritative sources before implementation.
2. **No Liability:** This system is provided as an educational tool to support—not replace—the user's own analytical process. The authors and contributors are not liable for inaccuracies resulting from the use of this framework.
